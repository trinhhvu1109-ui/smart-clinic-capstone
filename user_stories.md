# User Stories

## Doctor
### 1. View Daily Appointments
- **Title:** View Daily Appointments
- **Criteria:** As a doctor, I want to see all appointments for today so that I can prepare before meeting patients.
- **Priority:** High
- **Story Points:** 3

### 2. Update Availability
- **Criteria:** As a doctor, I can set my available time slots for patients to book appointments.
- **Priority:** Medium
- **Story Points:** 2

## Patient
### 1. Book Appointment
- **Title:** Book Appointment with Doctor
- **Criteria:** As a patient, I want to book an appointment with a doctor to receive treatment.
- **Priority:** High
- **Story Points:** 3

### 2. View Prescription
- **Criteria:** As a patient, I can see my prescriptions after a visit.
- **Priority:** Medium
- **Story Points:** 2

## Admin
### 1. Manage Doctors
- **Title:** Add/Update/Delete Doctor
- **Criteria:** As an admin, I can manage doctor accounts to keep the system updated.
- **Priority:** High
- **Story Points:** 3

### 2. Generate Reports
- **Criteria:** As an admin, I can generate daily appointment reports by doctor.
- **Priority:** High
- **Story Points:** 2
